美佳のタイプトレーナ 記号編 Ｖｅｒ２．０２．０１ JAVASCRIPT版ソースコード

　教育用のタイプ練習ソフト「美佳タイプトレーナ 記号編 V2.02」のJAVASCRIPT版ソースコードです。
　ブラウザーで作動します。
　Chrome、Edge、Firefoxで動作確認済です。
　Chromebookでも動作します。
　著作権を表示する初期画面は表示されません。
　成績はクッキーに保存されます。保存期間は一年です。
　成績はサーバーには保存されません。
　オフラインでも作動しますがオフラインで実行した場合は成績は保存されません。
　Windows版では練習時間記録ファイルmikakigo.logと最高速度記録ファイルmikakigo.spdが保存されますが、
　このJAVASCRIPT版ではmikakigo.logとmikakigo.spdは保存されません。
　Windows版では合計練習時間はプログラムを起動してから終了するまでの時間ですがJAVASCRIPT版では個別の累積練習時間の合計です。
　Windows版では実行時にウィンドーのサイズを変更できますが、このJAVASCRIPT版では実行時にサイズの変更はできません。
　ウィンドーのサイズを変更したい場合はindex.htmlのキャンバスサイズを変更して下さい。
　メニューの項目に「終了」はありません。終了したいときはウィンドーをクローズして下さい。

　このソースコードの著作権は今村二朗が所有します。
　改変、転載、他ソフトの使用など自由にお使いください。

注意事項

１．
キーボードの種類によっては記号の配列が異なる場合があります。

２.
チルダー記号 「~」 は 「～」 で表示されます。

３.
バックスラッシュキー「＼｣キーを押すと「￥」記号が表示されますがバグではありません。
